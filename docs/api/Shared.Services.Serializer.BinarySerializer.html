    Class BinarySerializer

    A binary serializer for IBinarySerializable objects.

    Remarks

    BinarySerializer and IBinarySerializable work with SerializedData, which requires IConvertible values.
    It encodes IConvertible values with optional un-encoded tags.

    To read back tagged values, (assuming BinaryReader reader): 

    var readTag = reader.readString();
             var readValue = (T)BinarySerializer.ReadConvertible(reader, typeof(T));

    While reading back untagged values requires only: 
    var readValue = (T)BinarySerializer.ReadConvertible(reader, typeof(T));
