    Interface ITypeRegister<T>

    Registers Types with objects by RegistryRelation, enabling look-up at runtime.

    E.g. the Type : typeof(TroopCard), if registered with "TroopCard", can be retreived by:

    Type troopCardType = Registry["TroopCard"];
    where Registry is an instance of TypeRegister.

    Namespace: Shared.Services.Registry
    Assembly: Model.dll
    Syntax
