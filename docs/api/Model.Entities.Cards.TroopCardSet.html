    IsValidTrade(ICard[])

    Determines whether a group of cards is a valid set for trade-in.

    Declaration
    public bool IsValidTrade(ICard[] cards)
    Parameters
    Type Name Description
    ICard[] cards

    An array of cards to test.
    Returns
    Type Description
    bool

    true if the collection cards qualifies exactly as a valid trade-set. Otherwise, false.
    Remarks

    TroopCard, as the default card set, stipulates a matching set of ICard:
    (1) contains three cards
    (2) contains only tradeble cards (see IsTradeable)
    (3) contains TroopCards with all identical OR all different TroopInsignia (after wilds).
