About Hazard!

Hi! I'm Josh McKnight, an aspiring software developer.

I'm proud to present Hazard!: Nothing ventured, nothing gained, a local, "hot-seat" game for two to six players who vie for military control of the globe.

Thanks to the art of my lovely wife, Kiah, your conquests come with real charm!

Hazard! was developed using Windows Presentation Foundation (WPF).

Deployed as an MSIX package (.msixbundle), Windows users can get the application by clicking the download button. For more installation details, please visit the Download section.

Hazard! emulates a popular board game's ruleset. See the "Help" and "About" sections of the application menu after installation.

Warning!

This is a demo of software development, not game development or design.

Don't expect too much fun.

Still, you can skip tedious setup by loading a demo save file!

Background

In high school I had one programming project I could never get fully working: a "Risk" clone in VB6. When I decided to return to software development, I figured I should finish what I had started. Of course, VB6 wouldn't serve any more. The program would have to be written in a contemporary language, conform to modern industry standards and practices, and leverage current technologies. Hopefully, developing it would also begin to teach me how to work within a team.

This is that project.
