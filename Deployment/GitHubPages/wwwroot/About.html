Thanks to the art of my lovely wife, Kiah, your conquests come with real charm!

About Hazard!

Hazard! was developed using Windows Presentation Foundation (WPF).

Deployed as an MSIX package (.msixbundle), Windows users can get the application by clicking a download button above. For more installation details, please visit the Download page.

Hazard! emulates a popular board game's ruleset. See the "Help" and "About" sections of the application menu after installation.

For full details on the project, see the rest of this site (especially the GitHub Repo)!
