About Hazard!

Hazard!: Nothing ventured, nothing gained is a strategic conquest game for two to six players who vie for military control of the globe.

And thanks to art by my wife, Kiah, you get a touch of charm with your campaign!

Hazard! was developed using Windows Presentation Foundation (WPF) and deployed as an MSIX package (.msixbundle). Windows users can get the application by clicking the download button.

Hazard! emulates a popular board game's ruleset. See the "Help" and "About" sections of the application menu after installation.

Background

In high school, I had one programming project I could never get fully working: a "Risk" clone in VB6. When I decided to pivot back to software development, I figured I should finish what I had started.

Of course, VB6 wouldn't serve any more. The program would have to be written in a contemporary language, conform to modern industry standards and practices, and leverage current technologies. Hopefully, developing it would also teach me something about how to work within a team.

This is that project, now evolved into a software engineering showcase!

Warning!

This is a demo of software development, not game development or design.

Don't expect too much fun.

Still, you can skip tedious setup by loading a demo save file!

✨ Key Engineering Highlights

🎭 State Machine & Events

Regulated game flow with event-driven architecture enabling loose coupling between components.

💾 Custom Serialization

Binary serialization framework with reflection-based prototyping and production optimization paths.

🗂️ Type Registry System

Dynamic type resolution with data-driven configuration supporting runtime modularity.

☁️ Azure DevOps Pipeline

Complete CI/CD with multi-architecture builds, code signing, and secure distribution.

🚀 Deployment & Distribution

📦 Multi-Architecture MSIX

Self-contained packages for x64 and ARM64 with Microsoft-verified code signatures via Azure Trusted Signing.

🔒 Secure Distribution

Time-limited download links with escalating rate limiting and persistent ban management prevent abuse.

⚙️ CI/CD Pipeline

Automated Azure DevOps pipeline with federated authentication and cost-optimized storage management.
