Hi! I'm Josh McKnight. I'm an aspiring software developer.

I'm proud to present Hazard!: Nothing ventured, nothing gained, a local, "hot-seat" game for two to six players vying for military control of the globe. And thanks to the art of my lovely wife, Kiah, players can collect cards with some real charm!

About Hazard!

Hazard! was developed using Windows Presentation Foundation (WPF).

Deployed as an MSIX package (.msixbundle), Windows users may install the application by clicking a download button above. For more installation details please visit the Download page.

Hazard! emulates a popular board game's ruleset. See the "Help" and "About" sections of the application menu.

For full details on the project, please peruse the site and don't forget the GitHub Repo!
